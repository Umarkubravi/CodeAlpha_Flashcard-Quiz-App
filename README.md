# Flashcard Quiz App

## Overview
Flashcard Quiz App is a simple and interactive web application built using HTML, CSS, and JavaScript. It helps users study by creating, updating, deleting, and browsing flashcards.

## Features

- Display flashcards
- Show/Hide answers
- Previous and Next navigation
- Add new flashcards
- Update existing flashcards
- Delete flashcards
- Progress indicator
- Flashcard list
- Responsive design
- Local Storage support

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Open `index.html` in any modern web browser.
